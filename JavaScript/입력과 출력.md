# 1. 입력

## 사용자 입력값 받기

* `prompt()`함수는 사용자의 입력값을 받는 함수입니다.

```js
prompt();
```

함수 괄호 안에 원하는 내용을 적으면, 정보를 입력받을 때 사용자에게 원하는 정보가 무엇인지 알려줄 수 있습니다.

```js
prompt("나이를 입력하세요.");
```

또한, 텍스트 안에 기본값을 설정할 수도 있습니다.

```js
prompt("이름을 입력하세요.", "완고한 고집의 힘, 페리도트그린!");
```

# 2. 출력

## 알림창으로 출력하기 - `alert() 함수`

* `alert()`로 웹 브라우저로 알림을 보내 원하는 데이터를 출력할 수 있습니다.

```js
alert("안녕하세요!");
```

## 웹 브라우저 화면에 출력하기 - `document.write()` 함수

* 자바스크립트를 통해 웹 브라우저에 정보를 출력할 일이 많습니다. `document.write()`는 원하는 내용을 웹 브라우저에 출력합니다.

```js
document.write("깨지고 부서져라...");
```

사용자의 입력을 받아 출력할 수도 있습니다.

```js
var command = prompt("명령: ");
document.write(command + "라는 명령을 받았습니다.");
```

## 콘솔에 출력하기 - `console.log()` 함수

* 웹 브라우저에 출력하는 것이 아닌 콘솔에 출력할 수도 있습니다.

```js
var quote = prompt("띵언을 입력하세요");
console.log(quote + ", 그대의 말에 눈물을 흘렸읍니다...");
```
